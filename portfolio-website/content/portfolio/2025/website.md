## Overview

I created a personal website to serve as an online resume and comprehensive project portfolio. It is designed to showcase my experience, skills, and engineering work in an easily accessible format.

I intially created the site using Material for Mkdocs, as shown above in the header picture. I then transition to developing the site using Hugo and the Phantom theme from HTML5 UP.
